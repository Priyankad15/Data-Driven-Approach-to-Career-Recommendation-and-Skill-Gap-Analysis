# 🚀 Skill Bridge – AI-Powered Career Recommendation & Skill Gap Analysis

An intelligent **AI-powered Career Recommendation and Skill Gap Analysis Platform** that helps students and job seekers discover suitable career paths, identify missing skills, receive personalized learning recommendations, and build resumes for their dream careers.

---

## 🌐 Live Demo

🔗 **Live Application:** https://skill-bridge-bn53.onrender.com

💻 **GitHub Repository:** https://github.com/Priyankad15/Data-Driven-Approach-to-Career-Recommendation-and-Skill-Gap-Analysis

---

## 🚀 Overview

Skill Bridge empowers users to make informed career decisions using AI and data-driven insights.

Users can:

* 🎯 Discover the most suitable career paths
* 📊 Analyze skill gaps for their desired careers
* 🤖 Get AI-powered career guidance
* 📚 Receive personalized learning recommendations
* 📄 Build ATS-friendly resumes
* 💡 Explore hands-on mini projects
* 🏆 Practice coding through daily challenges
* 👤 Manage their career profile securely

## ⚡ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Priyankad15/Data-Driven-Approach-to-Career-Recommendation-and-Skill-Gap-Analysis.git

cd Data-Driven-Approach-to-Career-Recommendation-and-Skill-Gap-Analysis
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

### 4️⃣ Start the Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:5173
```
